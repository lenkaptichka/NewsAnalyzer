Об авторе

Меня зовут Ефанова Елена и я начинающий frontend-разработчик. За семь месяцев обучения в Яндекс.Практикуме мной были освоены технологии HTML5, CSS3, JavaScript ES6, GIT и Webpack.

Вёрстка данного проекта выполнена по макету Figma. Каркас макета реализован на Flex layout и Grid layout. Также в данной работе используются семантичекие теги и валидация при помощи регулярных выражений. Применяется методология БЭМ для именования классов и структурирования файлов. JS-код написан с использовалось ООП. Также в работе применяются два внешних API (один для поиска новостей, второй - для подгрузки коммитов).
